## Clone the repo and install dependencies
```bash
git clone https://github.com/shibnev/webpack-boilerplate.git
cd webpack-boilerplate
npm i
```
## Usage

### Development server

```bash
npm run dev
```

### Production build

```bash
npm run build
```
